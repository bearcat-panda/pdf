### 1.使用说明 [json转excel]

```shell
Mac： 使用说明
$ ./main -i 输入文件目录 -o 输出文件目录
Liunx：
$ main -i 输入文件目录 -o 输出文件目录
Window:
$ main.exe -i 输入文件目录 -o 输出文件目录
```
